# 订单管理系统需求文档

## 1. 订单创建

### 1.1 下单流程
1. 用户选择商品加入购物车
2. 进入购物车页面
3. 确认订单信息
4. 选择收货地址
5. 选择支付方式
6. 提交订单

### 1.2 库存检查
下单时需要检查商品库存，库存不足时提示用户。

### 1.3 订单金额计算
订单金额 = 商品金额 + 运费 - 优惠金额

### 1.4 订单超时
订单创建后，多久未支付会自动取消？（需要确认）
