```
│  └─────────────────────────────────────────┘     │
│                                                   │
│  错误提示区域（密码错误5次锁定30分钟）             │
└─────────────────────────────────────────────────┘
```

**交互要点：**
- "获取验证码"按钮点击后 60s 倒计时，期间禁用
- 手机号格式校验（11位、1开头、第二位3-9）
- 密码错误 5 次后账号锁定 30 分钟，显示倒计时
